# VictoriaMetrics Ops

VictoriaMetrics 운영 보조 자료를 두는 공간입니다.

| 폴더 | 내용 |
|------|------|
| `config/helm/` | 개발/HA용 Helm values |
| `config/kubernetes/` | VMCluster, VMAgent, VMAlert CRD 예시 |
| `config/prometheus/` | Prometheus remote_write 설정 예시 |

운영 개념과 절차 문서는 `docs/`에 두고, 실제 적용 가능한 매니페스트와 설정 예시는 `ops/`에 둡니다.

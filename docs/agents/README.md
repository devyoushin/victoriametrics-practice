# AI 작업 지침

이 디렉터리는 VictoriaMetrics 문서를 다룰 때 사용하는 AI 작업 지침을 모은다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| `doc-writer.md` | 문서 작성 기준 |
| `storage-designer.md` | 저장소/백업 설계 |
| `alerting-advisor.md` | 알림과 규칙 설계 |
| `troubleshooter.md` | 문제 해결 절차 |

## 언제 보나

- 새 문서를 쓸 때 `doc-writer.md`
- storage/backup을 설계할 때 `storage-designer.md`
- Alerting을 설계할 때 `alerting-advisor.md`
- 장애 분석이나 재현이 필요할 때 `troubleshooter.md`

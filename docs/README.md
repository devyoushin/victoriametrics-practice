# VictoriaMetrics Docs

VictoriaMetrics 문서는 학습 문서, 규칙, 템플릿, 에이전트 프롬프트를 함께 관리합니다.

| 폴더/문서 | 내용 |
|------|------|
| `install.md` | Helm, systemd, Docker Compose 설치 |
| `architecture-guide.md` | Cluster 컴포넌트와 데이터 흐름 |
| `storage-guide.md` | EBS 스토리지와 백업 |
| `vmagent-guide.md` | scraping과 remote_write 에이전트 |
| `grafana-datasource-guide.md` | Grafana 데이터소스 연동 |
| `recording-rules-guide.md`, `alerting-guide.md` | vmalert와 Alertmanager 운영 |
| `ha-guide.md`, `monitoring-guide.md` | 고가용성 및 메타 모니터링 |
| `troubleshooting-guide.md` | 문제 해결 |
| `rules/` | 문서 작성 및 운영 규칙 |
| `templates/` | 서비스 문서, 런북, 장애 보고서 템플릿 |
| `agents/` | Claude 에이전트 프롬프트 |

처음 읽을 문서는 `install.md`입니다.
